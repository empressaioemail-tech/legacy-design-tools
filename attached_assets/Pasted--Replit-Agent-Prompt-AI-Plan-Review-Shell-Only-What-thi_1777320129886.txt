**Run this AFTER prompt 01.** It assumes the monorepo and `@legacy/portal-ui` package already exist.

**How to use:** in the same Replit (the one created by prompt 01), open the Replit Agent and paste everything between the `BEGIN` and `END` markers below.

**After it finishes:** the design-tools app and the plan-review app should be runnable simultaneously. Open both URLs side-by-side in browser tabs.
