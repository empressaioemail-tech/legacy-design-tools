# Cortex — Logo Pack

The brand mark is **`[ cortex ]`** set in JetBrains Mono. Lightweight brackets (weight 300), medium wordmark (weight 500).

```
[ cortex ]
```

## The font

The logo uses **JetBrains Mono** (free, [Google Fonts](https://fonts.google.com/specimen/JetBrains+Mono)).
Weights used: **300** (brackets) and **500** (wordmark).

Add this to your site `<head>` once — every SVG below renders correctly afterward:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@300;500&display=swap" rel="stylesheet">
```

The SVGs fall back gracefully to `ui-monospace, SF Mono, Menlo, Consolas, monospace` if JBM isn't available — but loading it gives the intended look.

## Web usage

### Light theme (dark mark on light bg)

```html
<img src="/cortex-logo-pack/svg/cortex-wordmark-light.svg" alt="Cortex" height="32" />
```

### Dark theme (light mark on dark bg)

```html
<img src="/cortex-logo-pack/svg/cortex-wordmark-dark.svg" alt="Cortex" height="32" />
```

### Auto-adapting (recommended)

Use the `currentColor` variant inline and control with CSS:

```html
<span class="brand">
  <!-- paste the contents of svg/cortex-wordmark.svg here -->
</span>

<style>
  .brand svg { color: #0d0d0d; height: 32px; }
  @media (prefers-color-scheme: dark) {
    .brand svg { color: #fafaf7; }
  }
</style>
```

### Favicon

```html
<link rel="icon" type="image/svg+xml" href="/cortex-logo-pack/svg/favicon.svg" />
<link rel="apple-touch-icon" href="/cortex-logo-pack/png/apple-touch-icon-180.png" />
```


## Clear space & sizing

- Reserve clear space equal to the **height of one bracket** on all sides of the wordmark.
- Minimum render height: **20px** (wordmark), **16px** (icon).
- For favicons and tab icons, use `favicon.svg` (heavier weight, tighter spacing — designed for sub-32px display).
